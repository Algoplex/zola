# OpenGPT

[opengpt.chat](https://opengpt.chat)

**OpenGPT** is the open-source chat interface for all your models.

## Features

- Multi-model support: ChatGPT, OpenAI, Mistral, Claude, Gemini
- File uploads
- Clean, responsive UI with light/dark themes
- Customizable: user system prompt, multiple layout options
- Full MCP support (wip)
